5x5 Numerical Game
ASS1

Language:
C# (.NET)

Source Files:
- Program.cs
- GameUI.cs
- GameEngine.cs
- GameSaveManager.cs

Compilation:


How to Run:
Ass1.exe [optional_save_file]

Game Rules:
- 5x5 board
- Place number in ascending order starting from 1to 25
- +1 point if the new placed number is one away from the last number.
- Invalid move ends game immediately
- No undo
- Available to save a txt file of the board.
